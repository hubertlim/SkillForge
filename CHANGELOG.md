# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-03-28

### Added

- Visual drag-and-drop canvas with React Flow
- 11 predefined skill blocks across 6 categories
- Editable node properties (label, description, instructions)
- Topological export to valid SKILL.md with YAML frontmatter
- Copy to clipboard and file download
- Dark theme UI
- Docker support
- Community skills directory with TDD workflow example
