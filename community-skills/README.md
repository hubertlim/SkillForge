# Community Skills

A collection of skill workflows built and shared by the SkillForge community.

## How to submit

1. Create a folder with your skill name (e.g. `api-design/`)
2. Add your exported `SKILL.md` inside it
3. Optionally add a `README.md` describing the workflow
4. Open a PR with the `community-skill` label

## Available skills

| Skill | Description |
|-------|-------------|
| [tdd-workflow](tdd-workflow/) | Test-driven development workflow: brainstorm → plan → TDD → implement → review |

Want yours listed here? Open a PR.
